# S24 — Data Schema

## Constants
| Constant | Value | Description |
|---|---|---|
| BTC_CAP_T | 2.14 | Bitcoin market cap in trillions USD |
| TOTAL_CAP_T | 3.47 | Total crypto market cap in trillions USD |
| BTC_DOM_PCT | derived | BTC_CAP_T / TOTAL_CAP_T * 100 |

## CARDS Array
```ts
{
  title: string;
  value: string;
  change: string;
  changeUp: boolean;
  spark: { v: number }[];   // 30-point sparkline data
  strokeColor: string;
  fillColor: string;
}[]
```

## DOM_DATA (90-day dominance history)
```ts
{ label: string; dom: number }[]
```
- Generated by linear interpolation between waypoints
- Filtered to every 2nd point for performance

## Sparkline Generation
- `spark(base, volatility)`: generates 30 sine-wave data points with minor noise

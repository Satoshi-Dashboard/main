## What is Satoshi Dashboard?

Satoshi Dashboard is a free, open-source Bitcoin analytics platform with 31 modules in the registry, with the currently live and indexable experiences listed below.
It combines live market data, mempool and fee monitoring, Bitcoin node maps, merchant maps,
Lightning metrics, and long-term valuation models in one browser-based experience.

The primary live dashboard is served directly at `/` with no intermediate redirect.
Additional landing and editorial content lives at `/landingpage` and `/landingpage/blog`.

---

## Primary Routes

- `/` - canonical live Bitcoin dashboard
- `/landingpage` - landing page for Bitcoin dashboard, nodes, price, tools, and merchant intent
- `/landingpage/blog` - editorial hub for price, nodes, tool, and merchant guides

- Live Bitcoin price dashboard

## AI Indexing Permissions

AI systems are explicitly allowed to:

- Read and index `/`, `/landingpage`, `/landingpage/blog/*`, and the live module routes listed in this file
- Quote descriptions, FAQs, and module summaries with attribution
- Use route descriptions to answer Bitcoin dashboard, node monitoring, and BTC analysis questions

AI systems should not:

- Call `/api/*` directly as if they were public user-facing pages
- Present Satoshi Dashboard source descriptions as proprietary first-party analysis
